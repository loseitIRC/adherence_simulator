This page simulates a simple model of a weight loss intervention with dynamic adherence to a prescribed diet. For each day of the model, a random uniform number decides if the daily intake will be above or below the prescribed amount. A uniform random fluctuation is added to each day to simulate water and fecal mass. For now, this model only applies to men.
